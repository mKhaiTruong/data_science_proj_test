# Simple end 2 end data science project

## Workflow:

1. Data ingestion and validate data
2. Data transformation - Feature Engineering, Data Preprocessing
3. Train & eval models
4. Simple Flask app

## Steps:

1. update config.yaml
2. update schema.yaml
3. update params.yaml
4. update the entity
5. update the configuration manager in src config
6. update the components
7. update the pipeline
8. update main.py

## Flask app running:

1. Run "python app.py" in the terminal, and a localohost appears. 
2. Type "http://127.0.0.1:5000/training" to train model if you have not done so.
3. back to "http://127.0.0.1:5000", fill in the form, and press "Predict"
